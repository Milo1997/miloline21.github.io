Geospatial Data Analysis and Map Projection

This map was created using ArcPro. The goal of this project was to evaluate the change in the burn scar after the devastating 2016 Yarloop fire. This map uses satellite data taken from the Landsat 8 database. The map took into account the land use categories and environmental phenology.
