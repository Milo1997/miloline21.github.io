Unsupervised Land Classification Modelling

This map categorises a section of the Bom Fortuna national park in Brazil that has been subject to severe deforestation. This was created using a Kmeans unsupervised classification model to evaluate the seperation between tropical forest cover and land cleared for agricultural purposes. This map was part of a larger remote sensing project evaluating the efficacy of land surface temperature as an indicator for land use change in difficult to access locations.

Workflow: Set boundbox geometry > Define temporal range > access and filter Landsat 8 imagery and sort by 'cloud cover' > create true composite RGB layer > rescale spectral reflectance and and set visual parameters > Create training data using K means (2 variables).
